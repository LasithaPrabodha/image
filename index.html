About SceneryX

ScenaryX is a application that uses machine learning algorithms to process and Identify what a given image of a scenery is about. for example this appication will be able to correctly identify whether a given scenery is an image of a beach,mountain,waterfall,sunset ect..

Grouping images into meaningful categories using low-level visual features is a challenging and important problem in content-based image retrieval and classification. Humans look at images and easily to identify the objects and scenarios in the images, using the knowledge and experience of what they have seen and gained thorough their lifetime but computer programs cannot clearly identify what the images represents and what the contents of the images are. this is the main problem we are trying to solve in this research project. the proposed program will segment a given image/s (related to travel/tourism) and from the segments create objects then classify the objects, and from the objects that relate to each other, identify the context of the given image/s and classify them accordingly. the program will be able to output a classified set of images or when given an image, describe the given image, specify what kind of image it is, indoor/outdoor, nature, city etc.
